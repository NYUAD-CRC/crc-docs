HPC Training¶

Important

We request all the users to pass the following quiz before submitting jobs, Make sure to log in your NYU Google Apps Account (e.g., NYU Google Mail) first! Otherwise, your access to the Quiz will be denied (Quiz)

There are essentially four important components a HPC user needs to be familiar with for successfully using it for computational research.

Component

Description

System

More about the system hardware and how to access it.

Storage

Page about user quotas and different types of storage available ($WORK, $SCRATCH, $HOME, $ARCHIVE).

Software

More about the software stack and modules, and what it offers you in terms of your application (MPI,Python,R,Singularity)
